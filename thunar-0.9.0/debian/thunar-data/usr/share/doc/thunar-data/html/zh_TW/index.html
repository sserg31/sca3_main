介紹 Thunar

Thunar 是 Xfce 桌面環境嶄新的檔案總管。 Thunar 以快速且易於使用為設計的原則。它的使用介面清爽又直覺，也沒有什麼奇奇怪怪的東西或預設一些無用的額外功能。Thunar 在啟動程式及載入目錄時的反應時間都奇快無比。

Thunar 檔案總管能夠讓您很直覺地管理自己的檔案與程式。您可以使用檔案總管從事以下的操作:

  • 建立目錄與文件
  • 檢視您的檔案和資料夾。
  • 管理您的檔案與資料夾。
  • 可以執行並管理自定的動作。
  • 存取隨身型的儲存媒體。
